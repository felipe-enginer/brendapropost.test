# Imagens do site

## Foto da profissional (obrigatória)
Adicione a sua foto de apresentação neste diretório com o nome exato:

```
images/brenda.jpg
```

Ela aparece no **hero** (primeiro contato) e na seção **Sobre**.
Use uma foto vertical (retrato), de preferência com boa iluminação e fundo neutro
— exatamente como a identidade visual do perfil
[@brendamartins.clinic](https://www.instagram.com/brendamartins.clinic).

Enquanto a foto não for adicionada, o site mostra automaticamente um
placeholder elegante com a marca (`placeholder-portrait.svg`).

## Imagens dos serviços
As fotos dos cards de serviço carregam de um banco de imagens online.
Para usar fotos reais dos seus procedimentos, substitua os endereços `src`
em `index.html` por arquivos locais, por exemplo:

```
images/sobrancelhas.jpg
images/micro-labios.jpg
images/remocao-tatuagem.jpg
images/manicure.jpg
images/pedicure.jpg
images/spa.jpg
```
